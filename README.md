# PL-25-26

## Descrição

Projeto desenvolvido no âmbito da unidade curricular de Processamento de Linguagens.

O objetivo do projeto consistiu na implementação de um compilador para uma subset da linguagem Fortran 77, traduzindo programas fonte para código executável na máquina virtual stack-based EWVM.
